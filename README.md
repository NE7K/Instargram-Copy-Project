# Instargramcopy

Instargramcopy 프로젝트는 Instagram의 주요 기능을 복사하여 구현한 프로젝트입니다.

## 핵심 기능
- 사진 업로드
- 게시물 작성
- 좋아요 및 댓글 기능

## 설치 및 실행 방법

1. 저장소를 클론합니다:
    ```bash
    git clone https://github.com/yourusername/instargramcopy.git
    cd instargramcopy
    ```

2. 필요한 패키지를 설치합니다:
    ```bash
    flutter pub get
    ```

3. 애플리케이션을 실행합니다:
    ```bash
    flutter run
    ```
